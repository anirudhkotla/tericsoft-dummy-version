# TericSoft Company Page

A dummy version of the TericSoft company page.

## Features

- Home page with hero section and features
- About Us page with company story and team information
- Projects page with featured projects
- Contact Us page with a form to send messages

## Setup

1. Clone the repository:
    ```sh
    git clone https://github.com/anirudhkotla/tericsoft-dummy-version.git
    ```
2. Open `index.html` in your browser to view the home page.

## Deployment

This project is deployed using GitHub Pages. To deploy your own version:

1. Push your changes to the `main` branch.
2. Go to the GitHub repository settings.
3. Scroll down to the GitHub Pages section.
4. Select the `main` branch as the source.
5. Save and wait for the deployment to complete.
